# Face-API.js Models

This directory contains the pre-trained models required for facial emotion detection using face-api.js.

## Models Included

1. **Tiny Face Detector** - Fast face detection model
2. **Face Landmark 68** - 68-point facial landmark detection
3. **Face Recognition** - Face recognition model
4. **Face Expression** - Emotion recognition (happy, sad, angry, fearful, surprised, disgusted, neutral)

## Files

- `tiny_face_detector_model-*` - Tiny face detector model files
- `face_landmark_68_model-*` - Facial landmark detection model files
- `face_recognition_model-*` - Face recognition model files
- `face_expression_model-*` - Emotion detection model files

## Source

These models are from the [face-api.js](https://github.com/justadudewhohacks/face-api.js) project.

## License

These models are provided by the face-api.js project and follow their respective licenses.

## Total Size

Approximately 7.1 MB

## Usage

These models are automatically loaded by the facial detection feature in the application. No manual intervention is required.
